python 线性代数：[3]矩阵转置

2015年07月15日

矩阵的转置很简单，就是将矩阵的行变为列，将列变为行，我们先通过例子看一下矩阵转置是怎么做的。然后验证几个规律。

  • 先创建一个矩阵A
  • 我们使用属性T来得到矩阵A的转置矩阵
  • 我们验证第一个性质：(A’)’=A
  • 再创建两个尺寸相同的矩阵
  • 验证矩阵转置的第二个性质：(A±B)’=A’±B’
  • 验证矩阵转置的第三个性质：(KA)’=KA’
  • 验证矩阵转置的第四个性质：(A×B)’= B’×A’

  • 本文用到的所有代码如下：

  • A

  • array([[1, 2, 3],

  • [4, 5, 6]])

  • A.T

  • array([[1, 4],

  • [2, 5],

  • [3, 6]])

  • A.T.T

  • array([[1, 2, 3],

  • [4, 5, 6]])

  • B

  • array([[1, 4],

  • [2, 5],

  • [3, 6]])

  • D

  • array([[0, 3],

  • [1, 4],

  • [2, 5]])

  • (B+D).T

  • array([[ 1, 3, 5],

  • [ 7, 9, 11]])

  • B.T+D.T

  • array([[ 1, 3, 5],

  • [ 7, 9, 11]])

  • 10*A.T

  • array([[10, 40],

  • [20, 50],

  • [30, 60]])

  • (10*A).T

  • array([[10, 40],

  • [20, 50],

  • [30, 60]])

  • array([[14, 32],

  • [32, 77]])

  • [22, 29, 36],
